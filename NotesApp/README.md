# Student Dashboard - Documentation ## Project Overview Student Dashboard is a modern web application built using **React + Vite + TypeScript**, styled with **Tailwind CSS + ShadCN** and powered by **Redux Toolkit** for state management. It provides students with a simple and interactive platform to manage assignments, subjects, and preferences while offering dark/light mode theming. --- ## Key Features ### Dashboard Management - Manage subjects and assignments - View assignment lists and deadlines - Organize subjects with tags ### Theming - ðŸŒ— Dark/Light theme toggle - Theme persistence using `localStorage` - Applied header, footer components - TailwindCSS utilities with `clsx` + `tailwind-merge` - Responsive layouts for desktop and mobile --- ## Dependencies ### Core - React - UI Library - TypeScript - Type development - Improve maintainability ### React - Functional components with Hooks - Context API for theme state - React Router for navigation ### Tailwind CSS + ShadCN - Utility-first styling - Pre-built accessible components - Dark/Light mode support - Fully customizable design system ### Redux Toolkit - Predictable global state management - Slice-based modular structure - Easy debugging
